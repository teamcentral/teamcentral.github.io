Code View ​

At the top level of the Data Hub, and for each individual map, you can flip to Code View, which shows the configuration that will feed into the mapping engine.

Advanced Configuration

Code View can be used for advanced features that the UI does not currently support, but that can be configured by the TeamCentral support team.

INFO

A Data Hub in Draft status can be modified and saved within the View Code feature.

View and Edit the Data Hub Code ​

  1. Select Build > Data Hubs from the main menu.

  2. Select the appropriate Data Hub; then select Actions > View Code.

    NOTE

    You can select Actions > View Code from within an Endpoint as well.

  3. As needed, edit the code.

    TIP

    Select the Toggle Fullscreen icon to view a larger window. Select the icon again to return to the smaller window.

  4. Select Save to save any edits.

NOTE

To exit Code View without saving any changes, select Close; then, select OK to confirm.

Or, select Reset to clear any edits made but remain in Code View.
